## Cara Menjalankan

1. Install [XAMPP](https://www.apachefriends.org/) atau [Laragon](https://laragon.org/)
2. Taruh semua file di folder `htdocs/biodata/`
3. Pastikan `fotogua.jpg` ada di folder yang sama
4. Buka browser → `http://localhost/biodata/index.php`
